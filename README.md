# EDA-Salary-Data-Analysis

## Project Overview

This project focuses on performing Exploratory Data Analysis (EDA) on a salary dataset to understand compensation trends, job roles, company ratings, and other key factors affecting salaries.

The goal of this analysis was to extract meaningful insights from raw salary data and understand patterns that can help job seekers, recruiters, and analysts make data-driven decisions.

This project was originally developed in a Kaggle notebook and later structured for GitHub.

## Objectives

-Understand salary distribution across different roles

-Analyze impact of company ratings on salary

-Identify high-paying job roles

-Detect missing values and handle them appropriately

-Explore relationships between experience, company, and compensation

## Key Analysis Performed

1️⃣ Data Cleaning

Checked for missing values

Handled null values appropriately

Cleaned inconsistent company names

Converted salary columns into usable numeric format

2️⃣ Salary Distribution Analysis

Visualized salary spread

Identified median, average, and outliers

Compared salary ranges across job roles

3️⃣ Company Rating vs Salary

Explored correlation between company rating and compensation

Identified whether higher-rated companies actually pay more

4️⃣ Role-Based Insights

Identified top-paying job titles

Analyzed demand patterns across different roles

5️⃣ Experience-Based Trends

Compared entry-level vs experienced salary brackets

Observed how salary scales with experience

## Key Insights

-Certain technical roles show significantly higher median salaries.

-Company rating does not always strongly correlate with higher compensation.

-Salary distribution is right-skewed with noticeable outliers.

-Experience plays a major role in compensation growth.
